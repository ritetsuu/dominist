Nagi#4089 Discord Bot

Instructions and command lists for the Nagi Discord bot.

The Nagi#4089 bot is a multi-purpose tool designed for server administration, member tracking, and a custom point-based economy. It uses SQLite for data persistence and Discord Slash Commands for interaction.

📊 Points & Economy The points system is stored in a local SQLite database. Even if the bot restarts, the points remain. /checkpoints [@user]: Displays the current balance of the mentioned user. If no user is mentioned, it shows your own balance. ⚙️ Utility & Server Stats Tools for monitoring the bot and your community. /uptime: Shows exactly how many days, hours, and minutes the bot has been running since its last restart. /oldestusers : Lists the first members to ever join the server (up to 25). Great for identifying veteran members. /serverinfo: Displays a summary of the server, including member count and creation date. /avatar [@user]: Pulls the full-size profile picture of any member. 🎲 Fun & Randomization /random: Randomly selects one human member from the server. Useful for giveaways or picking "Member of the Day." /randomnumber : Generates a random integer between the two numbers provided.
